# Chapter 3 · Basic Arithmetic Operations

A focused chapter on numeric computation in Python: all seven arithmetic
operators, operator precedence, augmented assignment shortcuts, the most
useful built-in numeric functions, and the standard `math` module for
more advanced calculations.

---

## Files in this folder

| File | Topics covered |
|------|----------------|
| `arithmetic.py` | Arithmetic operators (`+` `-` `*` `/` `//` `%` `**`), operator precedence (PEMDAS / BODMAS), augmented assignment (`+=` `-=` `*=` `/=` `//=` `**=`), `abs()`, `round()`, `divmod()`, `pow()`, `math.sqrt()` `math.pow()` `math.ceil()` `math.floor()` `math.log()` `math.log2()`, `math.pi`, `math.e`, real-world problems (distance, compound interest, temperature conversion, geometry) |

---

1. **Header** – chapter, topic, description, and learning goals.
2. **Examples** – short, runnable snippets that demonstrate each concept.
3. **Tasks** – problems for you to solve; no solution is given here.
4. **`# WRITE YOUR SOLUTIONS BELOW THIS LINE`** – your personal workspace.
