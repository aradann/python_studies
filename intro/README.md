# Chapter 1 · Intro

An introduction to Python covering the very first things you need to know:
how to produce output, the fundamental data types, and how to work with text.

---

## Files in this folder

| File | Topics covered |
|------|----------------|
| `hello_world.py` | `print()` function, positional arguments, the `sep` and `end` parameters, printing expressions |
| `basic_types.py` | `int`, `float`, arithmetic operators (`+` `-` `*` `/` `//` `%` `**`), `type()`, `isinstance()`, type conversion (`int()`, `float()`, `round()`) |
| `strings.py` | String literals, escape characters, multi-line strings, indexing, slicing, `.upper()` `.lower()` `.strip()` `.split()` `.join()` `.replace()` `.find()`, f-strings, membership testing (`in`) |

---

## How each file is organised

1. **Header** – chapter, topic, description, and learning goals.
2. **Examples** – short, runnable snippets that demonstrate each concept.
3. **Tasks** – problems for you to solve; no solution is given here.
4. **`# WRITE YOUR SOLUTIONS BELOW THIS LINE`** – your personal workspace.
5. **`# ---- SOLUTIONS ----`** – reference answers, commented out.
